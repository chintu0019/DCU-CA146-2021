Task

Assume exactly one command-line argument, an integer n.

Write a Python script named TASK_NAME which writes a diagonal line as illustrated in the example to standard output.

Example invocation
$ python3 TASK_NAME 7
Example standard output
*
 *
  *
   *
    *
     *
      *
Tip

The first line consists of an asterisk.

The second line consists of a single space character and an asterisk.

The third line consists of two space characters and an asterisk.

And so on.

There are seven lines in all.
