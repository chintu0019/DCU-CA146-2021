Box

Task

Assume exactly one command-line argument, an integer n.

Write a Python script named TASK_NAME which writes a box in the format illustrated in the example to standard output.

Example invocation
$ python3 TASK_NAME 7
Example standard output
 #######
#       #
#       #
#       #
#       #
#       #
#       #
#       #
 #######
Tip

The first and last lines consist of a single space followed by seven boundary characters.

The seven middle lines start and end with single boundary characters, each with seven space characters in between.

The boundary character is a hash sign.
